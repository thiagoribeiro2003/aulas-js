Manipulando o DOM

Conceito Geral

O Modelo de Objeto de Documento (DOM) é uma representação estruturada do documento como uma árvore.

Ilustração do DOM

O DOM possui métodos que permitem acesso à árvore, para que eles possam alterar a estrutura, estilo e conteúdo do documento.

Exemplos diversos

Usamos HTML para estruturar e CSS para estilizar.
